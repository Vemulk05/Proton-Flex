The Proton Flex project, developed by the Proton Flexers team in ECE 36200 at Purdue University, is a system designed to help users monitor and improve muscle activation using EMG signals. It utilizes MyoWare 2.0 Muscle Sensors to detect activity from targeted muscle groups. Users are challenged to match predefined muscle activation patterns, while an RP2350 microcontroller analyzes their performance and provides feedback in the form of an accuracy-based score.

Designed with individuals experiencing nerve damage or other neuromuscular conditions in mind, Proton Flex offers an engaging and interactive way to support rehabilitation. By turning muscle training into a guided and trackable experience, the system helps users build control, track progress, and stay motivated over time.
